## Getting Started

To get started with the API:
1. Navigate to your [Developer Dashboard](https://arca.vercel.app/dashboard/developer).
2. Generate an API Key.
3. Authenticate your requests using the `x-api-key` header.

## API Architecture

The Arca API is RESTful and built around standard HTTP methods. All endpoints are scoped to your merchant account and data.
