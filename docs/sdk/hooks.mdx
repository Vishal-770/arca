## useMecha

The `useMecha` hook is the primary way to check a user's subscription status programmatically. It connects to the Mecha Indexer to provide real-time status updates.

### Usage

```tsx
import { useMecha } from 'mechapay-react';

function PremiumFeature() {
  const { status, remainingSeconds, loading, error } = useMecha(PLAN_ID, USER_ID);

  if (loading) return <div>Synchronizing with blockchain...</div>;
  
  if (status !== 'ACTIVE') {
    return (
      <div className="locked-state">
        <h3>This feature is locked</h3>
        <p>You need an active subscription to access this content.</p>
      </div>
    );
  }

  return (
    <div className="premium-content">
      <h2>Welcome to the Inner Circle</h2>
      <p>Time remaining: {Math.floor(remainingSeconds / 3600)} hours</p>
    </div>
  );
}
```

### Returns

| Value | Type | Description |
| :--- | :--- | :--- |
| `status` | `'ACTIVE' | 'EXPIRED' | 'NONE'` | The current subscription status. |
| `remainingSeconds` | `number` | Time left until the subscription expires. |
| `tierId` | `string | undefined` | The active subscription tier ID (backward-compatible). |
| `tierIds` | `string[]` | Array of all active subscription tier IDs under this plan. |
| `loading` | `boolean` | Whether the SDK is currently fetching data. |
| `error` | `any` | Any error encountered during the sync. |

## useMechaPerks

The `useMechaPerks` hook resolves and aggregates all perks, features, and active entitlements across all active subscription tiers.

### Usage

```tsx
import { useMechaPerks } from 'mechapay-react';

function PremiumDashboard() {
  const { perks, loading, error } = useMechaPerks(PLAN_ID, USER_ID);

  if (loading) return <div>Synchronizing entitlements...</div>;
  if (error) return <div>Error loading perks: {error}</div>;

  return (
    <div>
      {perks?.map((tier, i) => (
        <div key={i} style={{ marginBottom: '16px' }}>
          <h3>Tier: {tier.tierName} (Expires: {tier.expiryDate.toLocaleDateString()})</h3>
          <ul>
            {tier.features.map((feature, j) => (
              <li key={j}>
                <strong>{feature.title}</strong>: {feature.description}
              </li>
            ))}
          </ul>
        </div>
      ))}
    </div>
  );
}
```

### Returns

| Value | Type | Description |
| :--- | :--- | :--- |
| `perks` | `SubscribedTierDetail[] | null | undefined` | List of all active tier details from active subscription tiers. `undefined` while loading, `null` if no active subscription. |
| `loading` | `boolean` | Whether the SDK is currently fetching data. |
| `error` | `any` | Any error encountered during synchronization. |
