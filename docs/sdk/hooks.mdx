## useArca

`useArca(planId, userId)` fetches the subscription state for a user and keeps a client-side countdown timer. If you omit `userId`, the hook stays idle and returns `status: "NONE"`.

### Usage

```tsx
import { useArca } from "arca-react";

function PremiumFeature() {
  const { status, remainingSeconds, loading, error } = useArca(PLAN_ID, USER_ID);

  if (loading) return <div>Syncing subscription...</div>;
  if (error) return <div>Failed to load subscription: {error}</div>;

  if (status !== "ACTIVE") {
    return (
      <div className="locked-state">
        <h3>This feature is locked</h3>
        <p>You need an active subscription to access this content.</p>
      </div>
    );
  }

  return (
    <div className="premium-content">
      <h2>Welcome to the Inner Circle</h2>
      <p>Time remaining: {Math.floor(remainingSeconds / 3600)} hours</p>
    </div>
  );
}
```

### Returns

| Value | Type | Description |
| :--- | :--- | :--- |
| `status` | `'ACTIVE' | 'EXPIRED' | 'NONE'` | Subscription status. |
| `remainingSeconds` | `number` | Seconds remaining for the active subscription. |
| `tierId` | `string | undefined` | The last active tier ID (if available). |
| `tierIds` | `string[]` | All active tier IDs for the plan. |
| `loading` | `boolean` | Whether the hook is fetching data. |
| `error` | `string | null` | Error message, if any. |

## useArcaPerks

`useArcaPerks(planId, userId)` combines subscription status with plan metadata to resolve all active perks and features. It returns `null` when there is no active subscription.

### Usage

```tsx
import { useArcaPerks } from "arca-react";

function PremiumDashboard() {
  const { perks, loading, error } = useArcaPerks(PLAN_ID, USER_ID);

  if (loading) return <div>Syncing entitlements...</div>;
  if (error) return <div>Error loading perks: {error}</div>;

  return (
    <div>
      {perks?.map((tier, i) => (
        <div key={i} style={{ marginBottom: "16px" }}>
          <h3>
            Tier: {tier.tierName} (Expires: {tier.expiryDate.toLocaleDateString()})
          </h3>
          <ul>
            {tier.features.map((feature, j) => (
              <li key={j}>
                <strong>{feature.title}</strong>: {feature.description}
              </li>
            ))}
          </ul>
        </div>
      ))}
    </div>
  );
}
```

### Returns

| Value | Type | Description |
| :--- | :--- | :--- |
| `perks` | `SubscribedTierDetail[] | null | undefined` | `undefined` while loading, `null` if no active subscription, or a list of active tiers. |
| `loading` | `boolean` | Whether the hook is fetching data. |
| `error` | `string | null` | Error message, if any. |
