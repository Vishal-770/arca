We have provided a reference implementation of the Arca SDK in a dedicated test project. This project demonstrates how to combine the `ArcaPricingTable` with the `useArca` hook for a complete membership experience.

## View on GitHub

You can find the full source code for the test project here:
[demo-app](https://github.com/Vishal-770/arca/tree/main/demo-app)

## What's Inside?

The example project includes:
- **Vite + React + TypeScript** setup.
- **`ArcaProvider`** configuration.
- **`ArcaPricingTable`** with a custom "Cyber" theme.
- **`useArca`** implementation for a real-time status banner.

## Running Locally

To run the test project on your own machine:

1. Navigate to the folder:
   ```bash
  cd demo-app
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```

## Reference Code

Here is the core logic from the example app's `App.tsx`:

```tsx
  import { ArcaProvider, ArcaPricingTable, useArca } from "arca-react";

function MembershipStatus() {
  const { status, remainingSeconds, loading } = useArca(PLAN_ID, USER_ID);

  if (loading) return <div>Syncing...</div>;

  return (
    <div className="status-banner">
      Status: <span className="status">{status}</span>
      {status === 'ACTIVE' && (
        <span className="timer">Expires in: {remainingSeconds}s</span>
      )}
    </div>
  );
}
```
