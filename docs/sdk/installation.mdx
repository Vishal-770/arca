## Installation

Install the SDK using your favorite package manager:

```bash
npm install arca-react
# or
yarn add arca-react
# or
pnpm add arca-react
```

## Configuration

To use the SDK, you must wrap your application (usually in `App.tsx` or `main.tsx`) with the `ArcaProvider`. This component provides the necessary context for all other SDK components and hooks.

```tsx
import { ArcaProvider } from 'arca-react';

function App() {
  return (
    <ArcaProvider apiKey="mp_live_...">
      <MyRoutes />
    </ArcaProvider>
  );
}
```

### Provider Props

| Prop | Type | Description |
| :--- | :--- | :--- |
| `apiKey` | `string` | **Required.** Your Arca Developer API Key. |

*(Note: Custom appearance, styling, and dark/light mode configurations are passed directly to individual components like `ArcaPricingTable` via their `appearance` prop).*
