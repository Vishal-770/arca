## Installation

Install the SDK using your favorite package manager:

```bash
npm install arca-react
# or
yarn add arca-react
# or
pnpm add arca-react
```

## Configuration

Wrap your application (usually `App.tsx` or `main.tsx`) with `ArcaProvider` so hooks and components can access your API key.

```tsx
import { ArcaProvider } from "arca-react";

function App() {
  return (
    <ArcaProvider apiKey="arca_...">
      <MyRoutes />
    </ArcaProvider>
  );
}
```

### Provider Props

| Prop | Type | Description |
| :--- | :--- | :--- |
| `apiKey` | `string` | **Required.** Your Arca Developer API Key. |

## Server-side Usage

If you need to check subscription status outside React, use `ArcaClient`:

```ts
import { ArcaClient } from "arca-react";

const client = new ArcaClient("arca_...");
const status = await client.getSubscriptionStatus("0x...", "user_123");
```

*(Note: UI customization is passed via component props such as `appearance` on `ArcaPricingTable`.)*
