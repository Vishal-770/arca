## Installation

Install the SDK using your favorite package manager:

```bash
npm install mechapay-react
# or
yarn add mechapay-react
# or
pnpm add mechapay-react
```

## Configuration

To use the SDK, you must wrap your application (usually in `App.tsx` or `main.tsx`) with the `MechaProvider`. This component provides the necessary context for all other SDK components and hooks.

```tsx
import { MechaProvider } from 'mechapay-react';

function App() {
  return (
    <MechaProvider apiKey="mp_live_...">
      <MyRoutes />
    </MechaProvider>
  );
}
```

### Provider Props

| Prop | Type | Description |
| :--- | :--- | :--- |
| `apiKey` | `string` | **Required.** Your Mecha Pay Developer API Key. |

*(Note: Custom appearance, styling, and dark/light mode configurations are passed directly to individual components like `MechaPricingTable` via their `appearance` prop).*
