The Arca React SDK (`arca-react`) is the official UI + hooks layer for Arca Protocol. It wraps authentication, plan data fetching, subscription status, and checkout routing in a small set of components and hooks.

## Key Features

<CardGroup cols={2}>
  <Card title="Provider Pattern" icon="palette">
    App-level configuration via `ArcaProvider`.
  </Card>
  <Card title="Pricing UI" icon="table-columns">
    `ArcaPricingTable` renders a full pricing and checkout flow.
  </Card>
  <Card title="Feature Gating" icon="shield-check">
    Hooks for access control and entitlements.
  </Card>
  <Card title="Client API" icon="terminal">
    `ArcaClient` for server-side or scripted checks.
  </Card>
</CardGroup>

## Architecture

The SDK talks to the Arca API (default base URL: `https://api-arca.vercel.app`). It pulls plan metadata and optional subscription status from `/api/sdk/plan/:planId`, so you can avoid direct blockchain or subgraph calls in your UI.
