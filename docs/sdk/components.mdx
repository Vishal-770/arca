The Arca React SDK ships with a flagship pricing component plus a provider for configuration.

## Components

- **ArcaProvider**: App-level configuration context. See [ArcaProvider](/sdk/components/arca-provider).
- **ArcaPricingTable**: High-fidelity pricing and checkout UI. See [ArcaPricingTable](/sdk/components/arca-pricing-table).

## ArcaPricingTable

The `ArcaPricingTable` auto-loads plan data, renders tier cards, and routes users into checkout. If you pass a `userId`, it also highlights the active tier and shows remaining time.

### Usage

```tsx
import { ArcaPricingTable } from "arca-react";

function PricingPage() {
  return (
    <ArcaPricingTable
      planId="0x..."
      userId="user_123"
      recommendedTierId="tier_pro"
    />
  );
}
```

### Props

| Prop | Type | Description |
| :--- | :--- | :--- |
| `planId` | `string` | **Required.** The on-chain plan ID to display. |
| `userId` | `string` | **Optional.** Your internal user ID (used for subscription status + checkout tracking). |
| `redirectUrl` | `string` | **Optional.** Redirect back after checkout. Defaults to the current URL. |
| `appearance` | `ArcaAppearance` | **Optional.** Theme tokens and element-level style overrides. |
| `className` | `string` | **Optional.** Class name applied to the outer container. |
| `style` | `React.CSSProperties` | **Optional.** Inline styles for the outer container. |
| `recommendedTierId` | `string` | **Optional.** Tier ID to highlight. Defaults to the second tier. |
| `hideBranding` | `boolean` | **Optional.** Hide the "Powered by Arca" footer. |
| `customLabels` | `ArcaLabels` | **Optional.** Override button and status copy. |
| `classNames` | `ArcaClassNames` | **Optional.** Tailwind or class overrides for internal elements. |
| `renderHeader` | `(plan, subscription) => React.ReactNode` | **Optional.** Replace the default header area. |
| `renderFooter` | `() => React.ReactNode` | **Optional.** Replace the default footer. |
| `renderTierButton` | `(tier, state, handleSelect) => React.ReactNode` | **Optional.** Replace per-tier CTA buttons. |
