The Arca React SDK (`arca-react`) provides a professional, high-fidelity set of hooks and components to integrate the Arca Protocol into your React ecosystem.

## Installation

Install the package using your preferred package manager:

```bash
npm install arca-react
# or
yarn add arca-react
# or
pnpm add arca-react
```

## Peer Dependencies

Ensure you have the following peer dependencies installed (standard for most modern React projects):

```json
{
  "react": ">=18",
  "react-dom": ">=18"
}
```

## Basic Configuration

To use the SDK, you must wrap your root application component with the `ArcaProvider`. This initializes the connection to the Arca Protocol and ensures that all nested components can access your configuration.

```tsx
import { ArcaProvider } from 'arca-react';

function Root() {
  return (
    <ArcaProvider 
      apiKey="mp_live_..." 
    >
      <App />
    </ArcaProvider>
  );
}
```

<Tip>
  You can find your API Key in the [Arca Developer Dashboard](https://arca.vercel.app/dashboard/developer).
</Tip>
