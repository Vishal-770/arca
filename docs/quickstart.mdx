## Authentication

All API endpoints are authenticated using an API key passed in the headers.

```bash
curl -X GET "https://api-arca.vercel.app/api/v1/wallet" \
  -H "x-api-key: arca_..."
```

## Base URL

- Production: `https://api-arca.vercel.app/api/v1`
- Local dev: `http://localhost:3001/api/v1`

## Response Format

Success responses include `success: true` and endpoint-specific fields:

```json
{
  "success": true,
  "walletAddress": "0x123abc456def7890"
}
```

Error responses include `error` and `code`:

```json
{
  "error": "Unauthorized: Invalid or revoked API key",
  "code": "UNAUTHORIZED"
}
```
