---
title: "Plan Analytics"
api: "GET /api/v1/plans/{id}/analytics"
description: "Retrieve detailed on-chain statistics for a specific plan."
---

### Path Parameters

<ParamField path="id" type="string" required>
  The unique ID of the plan.
</ParamField>

### Headers

<ParamField header="x-api-key" type="string" required>
  Your Arca Developer API Key.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the request was successful.
</ResponseField>

<ResponseField name="analytics" type="object">
  <Expandable title="properties">
    <ResponseField name="planId" type="string">The ID of the plan.</ResponseField>
    <ResponseField name="active" type="boolean">Whether the plan is currently accepting subscriptions.</ResponseField>
    <ResponseField name="subscriptionCount" type="number">Total number of subscriptions for this plan.</ResponseField>
    <ResponseField name="totalGrossVolume" type="string">Total USDC processed by this plan.</ResponseField>
    <ResponseField name="totalFeesCollected" type="string">Total protocol fees paid by this plan.</ResponseField>
    <ResponseField name="netRevenue" type="string">Net revenue earned by the merchant.</ResponseField>
    <ResponseField name="lastSubscriptionAt" type="string | null">Unix timestamp of the last subscription.</ResponseField>
    <ResponseField name="createdAt" type="string">Unix timestamp when the plan was created.</ResponseField>
    <ResponseField name="updatedAt" type="string">Unix timestamp when the plan was last updated.</ResponseField>
    <ResponseField name="tiers" type="array">List of active tiers for the plan.</ResponseField>
  </Expandable>
</ResponseField>

```bash cURL
curl -X GET "https://api-arca.vercel.app/api/v1/plans/0x123.../analytics" \
  -H "x-api-key: arca_..."
```
</RequestExample>
