---
title: "Authentication"
description: "How to authenticate requests to the Arca API."
---

All API requests must include an API key in the `x-api-key` header. You can manage keys in the [Developer Dashboard](https://arca7.vercel.app/dashboard/developer).

## API Keys

API keys are tied to your merchant account and grant access to your plans, analytics, and subscriber data. Keys are prefixed with `arca_`.

<Warning>
  Never share your API keys or commit them to source control. Use server-side storage where possible.
</Warning>

## Using the API Key

```bash
x-api-key: arca_...
```

### Example Request

```bash
curl -X GET "https://api-arca.vercel.app/api/v1/wallet" \
  -H "x-api-key: arca_..."
```

## Security

If a key is compromised, revoke it in the dashboard and generate a new one. Revoked keys return `401 UNAUTHORIZED`.
